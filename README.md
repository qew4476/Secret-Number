### 操作步驟：
  1. 玩家輸入猜測的數字後，滑鼠點擊確定鍵，網頁下方會顯示提示。根據獲得的提示繼續作答，每作答一次就會扣一分。

  2. 直到分數歸零或回答正確，遊戲結束。

  3. 若想繼續遊玩，可以點擊「重來一次」的按鍵。  

</br>
