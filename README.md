# Secret-Number
### [終極密碼](https://qew4476.github.io/Secret-Number/)

猜正確數字的網頁遊戲

### 操作步驟：
* 玩家輸入猜測的數字後，滑鼠點擊確定鍵，網頁下方會顯示提示。根據獲得的提示繼續作答，每作答一次就會扣一分。
* 直到分數歸零或回答正確，遊戲結束。
* 若想繼續遊玩，可以點擊「重來一次」的按鍵。
